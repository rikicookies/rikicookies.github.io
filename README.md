# 🧠 Vault Técnico – Master README

Tu bóveda personal de conocimiento estructurado.

---

## 📌 Propósito del Vault

Este sistema existe para centralizar lo que **realmente vale** conservar: conocimiento aplicable, formatos reutilizables, estrategias técnicas y respuestas de alto valor.
Aquí se guarda lo que resuelve, enseña o puede escalar. Es tu cerebro externo, sistematizado.

---

## 🧠 Filosofía de Uso

> "No todo entra. Solo lo que usaría mañana, lo que explicaría con orgullo, o lo que necesito entender mejor."

El vault no es una papelera ni una colección interminable de ideas. Es una **selección quirúrgica**.
Cada archivo debe tener propósito, cada nota debe transformarte.

## ⚙️ Convenciones del Vault

* Carpeta por tema principal.
* Archivo `*-index.md` como entrada central.
* Subnotas nombradas secuencialmente o temáticamente.
* Etiquetado manual (`#etiqueta`) y enlaces cruzados `[[...]]`.
* Uso extendido de Templater y Dataview.

---

## 🛠 Cómo Crear Contenido

1. Crea carpeta temática (si no existe).
2. Crea archivo `nombre-index.md` como índice.
3. Usa plantilla de nota o formato según categoría (`Plantilla::Contenido`).
4. Añade al Master Index con ícono y ruta.
